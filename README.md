# MenuIQ Legal

Public legal pages for the MenuIQ iOS app, served via GitHub Pages.

- **Privacy Policy:** https://chamallox.github.io/menuiq-legal/privacy/

Terms of Use are covered by Apple's standard EULA, linked from within the app.

## Editing

Pages are plain static HTML. Edit `privacy/index.html` and push to `main`;
GitHub Pages redeploys automatically.
